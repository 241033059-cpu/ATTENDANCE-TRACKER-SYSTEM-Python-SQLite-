# ATTENDANCE-TRACKER-SYSTEM-Python-SQLite-
The Attendance Tracker System is a simple and efficient database-driven application developed using Python and SQLite. The main objective of this project is to digitalize the daily classroom attendance process by storing student details, marking attendance, and calculating attendance percentages automatically.
